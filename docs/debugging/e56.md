# E56 — BLE_PAIRING_FAILED

BLE pairing with the FKMTime Display failed.

## What happens

The inner pairing event loop breaks. The outer connection loop (`'outer`) continues, which means the device will attempt to reconnect and pair again. The error is recorded **once per boot**.

## Possible causes

- The FKMTime Display rejected the pairing request (e.g., it is bonded to another device and rejected a new bond).
- RF interference corrupted the BLE key exchange.
- The FKMTime Display firmware version is incompatible with the device firmware.

## How to fix

1. **Clear bonds on both sides** — Delete the bond from the FKMTime device (settings menu) and from the FKMTime Display. Then re-pair.
2. **Retry pairing** — Move both devices close together in a low-interference environment and retry from the BLE pairing menu.
3. **Update firmware** — Ensure both devices are on the latest firmware.
4. **Contact support** — Open an issue on [GitHub](https://github.com/FKMTime/firmware).
